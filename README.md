# Dungeon Edtech Game

A language learning dungeon crawler for elementary school students.
Players explore procedurally generated dungeons, collect gem fragments 
representing vocabulary and solve word puzzles to progress through locked doors.

## Concept

Built on the premise that engagement and pedagogy don't have to be separate.
The core loop mirrors spaced retrieval practice - encounter a word at the door,
find its components through exploration, return and reconstruct it.

Designed with real classroom experience - 10 years as an ALT in Japanese 
elementary schools informed every design decision around anxiety, pacing 
and teacher deployment.

## Current Status
Vertical slice in active development.

## Core Loop
Explore → Break pots → Collect gems → Solve door puzzle → Progress

## Planned Features
- Configurable language packs (Japanese, Estonian, French etc)
